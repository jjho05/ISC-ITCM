Redes de Computadoras

Clave: SCD-1021 | Créditos SATCA: 2-3-5

Caracterización de la Asignatura

Esta asignatura aporta al perfil del Ingeniero en Sistemas Computacionales la habilidad para diseñar, configurar y administrar redes de computadoras para crear soluciones de conectividad, aplicando normas y estándares vigentes. Integra la capacidad de conocer, analizar y aplicar los diversos componentes físicos y lógicos involucrados en la planeación, diseño e instalación de redes, mediante el análisis de fundamentos, estándares y normas.

Competencia General de la Asignatura

  • hub Diseña y elabora un proyecto de cableado estructurado aplicando normas y estándares vigentes para la solución de problemas de conectividad.

Temario y Competencias por Unidad

1. Introducción a redes de datos

Competencia: Analiza las características y clasificación de las topologías de redes para seleccionar la más adecuada de acuerdo a las necesidades.

  • 1.1 Orígenes y evolución.
  • 1.2 Conceptos básicos de redes.
  • 1.3 Clasificación de redes.
  • 1.4 Topologías de redes: físicas y Lógicas.

2. Normas y estándares de redes de datos

Competencia: Aplica normas y estándares vigentes, que permitan un correcto diseño de una red local.

  • 2.1 Modelo de Comunicación OSI.
  • 2.2 Modelo de Comunicación TCP/IP.
  • 2.3 Estándares IEEE 802.
    • 2.3.1. Estándares IEEE 802 para Redes Alámbricas.
    • 2.3.2. Estándares IEEE 802 para Redes Inalámbricas.
  • 2.4 Pilas de protocolos y flujo de datos.

3. Dispositivos de red

Competencia: Selecciona y configura los dispositivos adecuados para garantizar el funcionamiento correcto de una red local.

  • 3.1 Dispositivos de capa física.
  • 3.2 Dispositivos de capa de enlace.
  • 3.3 Dispositivos de capa de red.
  • 3.4 Dispositivos de capas superiores.

4. Cableado estructurado

Competencia: Analiza, diseña, y elabora un proyecto de cableado estructurado para proporcionar soluciones de conectividad, conforme a las normas y estándares vigentes.

  • 4.1 Normas y estándares.
  • 4.2 Componentes y herramientas de Instalación.
  • 4.3 Identificación y especificaciones.

5. Planificación y diseño de una red LAN

Competencia: Planifica y diseña redes de datos para la implementación de un proyecto de conectividad en las empresas, utilizando una metodología de trabajo.

  • 5.1 Memoria técnica
  • 5.2 Análisis de necesidades y requerimientos.
  • 5.3 Diseño lógico de la red.
    • 5.3.1 Direccionamiento IP.
  • 5.4 Diseño físico de la red.
    • 5.4.1 Sistema de cableado estructurado.
    • 5.4.2 Dispositivos de red.
    • 5.4.3 Servidores y estaciones de trabajo.
    • 5.4.4 Sistemas operativos de red y aplicaciones.
    • 5.4.5 Pruebas y liberación.

Software y Herramientas Propuestos

Para las prácticas se recomienda el uso de herramientas de diagnóstico y software de diseño para la planificación y diseño de redes.

  • Herramientas de diagnóstico de cableado.
  • Software de diseño de redes basado en CAD.
